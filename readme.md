# Golang Fiber using Postgres.

## Usage:

1. Clone the repo
```ZSH
git clone git@github.com:VictorBischoff/GoFiber-Postgres.git
```
2. Enter the repo directory
```ZSH
cd GoFiber-Postgres
```
3. Install dependencies
```ZSH
go mod tidy
```
4. run the fiber server, and access at http://localhost:1312
```ZSH
go run cmd/web/*.go
```
